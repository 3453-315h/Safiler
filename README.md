# How to start

## Clone files
```bash
$ git clone https://github.com/Bo0oM/Safiler.git
$ cd Safiler
```

## Run server

```bash
$ [sudo] pip install -r requirements.txt
$ python server.py
```

## Open PoC
Open `PoC.xhtm` or `PoC.webarchive` in Safari.
Copy on a USB flash drive and carry it with you :)
